# chinese-code-arithmetic
Арифметика на системе остаточных классов (СОК)
Этот проект представляет собой реализацию арифметики на основе системы остаточных классов (СОК), также известной как "китайский код". В системе используются независимые параллельные вычисления по модулям, что позволяет эффективно работать с большими числами и обеспечивает отказоустойчивость.

Структура проекта
educational/ — калькулятор для учебного режима с подробным выводом всех вычислений по модулям. Позволяет понять, как работает система остаточных классов на примерах с малыми числами.

practical/ — калькулятор для рабочего режима, оптимизированный для работы с большими числами (до 80 десятичных разрядов). Выполняет быстрые вычисления без детального протоколирования.

Tests/tests_for_CRT.py — тесты для проверки корректности работы реализации СОК.

CRT.py — основной модуль, содержащий класс CRTInteger для работы с числами в системе остаточных классов.

Test.py — тестирование производительности операций, измерение скорости работы разных конфигураций системы.

report.docx — полный отчет по проекту с теоретическими основами, анализом литературы, описанием реализации и результатами экспериментов.

Запуск
Для использования учебного калькулятора с подробным выводом вычислений:

text
python educational/interface.py
Для использования практического калькулятора для работы с большими числами:

text
python practical/interface.py
Для тестирования производительности:

text
python Test.py
Для проверки корректности работы системы остаточных классов:

text
Запустите тесты из папки Tests/
Особенности реализации
Проект реализует полный набор арифметических операций (сложение, вычитание, умножение, деление) в системе остаточных классов. Поддерживаются два типа конфигураций: с малыми модулями для обучения и с большими модулями для практических вычислений. Все операции выполняются параллельно по модулям, что теоретически ускоряет вычисления.

В учебном режиме система выводит детальный протокол вычислений, показывая преобразования чисел в остатки и выполнение операций по каждому модулю. В практическом режиме акцент сделан на производительность и работу с числами большой разрядности.

Подробное описание теоретических основ, анализ производительности и сравнение с традиционными системами счисления представлены в отчете report.docx.
